#### [](#파이썬의-다운로드-및-설치 "파이썬의 다운로드 및 설치")파이썬의 다운로드 및 설치

파이썬을 다운받으시려면, [python.org](https://www.python.org/)로 가셔서  
해당 OS에 맞는 파일을 받으시면 됩니다.  
![](/images/python-basic-01_1.png)

저는 윈도우 환경에 맞춰 설명드릴게요.  
![](/images/python-basic-01_2.png)  
이미지를 보시면 다음과 같이 선택할 수 있습니다.

*   `x86` - 32bit 운영체제
*   `x86-64` - 64bit 운영체제

---

*   `web-based` - 설치실행 파일만 받고, 나머지 설치 파일들은 웹을 통해 다운로드
*   `executable` - 설치에 필요한 모든 파일이 들어있는 설치형
*   `embeddable` - zip 파일로 압축을 해제하여 바로 실행할 수 있는 비설치형

현재 버전은 3.7x 까지 나와있으며, 저는 3.6x 버전을 설치하여 사용하였습니다.  
3.6버전 이상이라면 크게 문제는 없을 것 같네요.

![](/images/python-basic-01_3.png)

설치하실 때 Add Python 3.7 to PATH 를 체크하고 설치하시는 것을 권장합니다.  
파이썬을 PATH를 지정함으로써 명령 프롬프트(cmd)창에 python 명령을 어디서든  
사용할 수 있습니다.

텍스트 에디터는  
`비주얼 스튜디오 코드`,`Notepad ++`,`Sublime Text` 등이 있으니  
사용하시기 편한걸로 설치하셔서 사용하시면 되겠습니다.

저는 파이썬을 설치할때 기본적으로 설치되어있는 IDLE를 사용하였습니다.
