### [](#클래스와-객체 "클래스와 객체")클래스와 객체

우리는 `클래스(Class)`를 통해 `객체(Object)`를 만듭니다.  
가장 쉬우면서 많이 쓰는 비유로는 붕어빵틀(클래스)로 붕어빵(객체)를 만듭니다.  
클래스는 어떤 실체를 표현할 때 추상화라는 개념을 사용해서  
여러가지 `속성`들을 지정할 수 있습니다.  
예를들어 `사람`이라는 실체를 표현하려고 합니다.  
사람이 가질 수 있는 수십,수백만 가지의 속성이 존재할 수 있으나 우리는 용도에 맞게  
그들의 속성을 부여합니다. 예를 들어 `사람`이라는 실체가 `학생` 신분이라면  
우리는 그들에게 학년,반,성적 등의 속성을 부여하게 될 겁니다.  
만약 그들이 `환자`라면 증상,진료기록,주치희 등의 속성을 부여하게 되겠죠.  
이러한 속성들을 지정하게 되면 우리는 다수의 객체를 만들어 낼 수 있습니다.  
학교에 다니고 있는 모든 학생들을 `학생`이라는 클래스를 통해 각각 만들 수 있고,  
소아과에서 진료를 받는 모든 아기들의 객체도 만들어 낼 수 있겠죠.

위와같이 클래스를 통해 객체를 만들어 사용하는 프로그래밍 방식을 `객체지향프로그래밍(OOP)`이라고 합니다.  
위의 예제가 OOP의 전부를 설명하진 못하지만 이 정도만 짚고 넘어가도록 하겠습니다.

#### [](#클래스-기본-구조 "클래스 기본 구조")클래스 기본 구조

{% codeblock %}
# 클래스 정의  
 class <클래스명>:  
 <클래스 내용>  
# 클래스 생성  
 <인스턴스 이름(변수)> = <클래스명>()  
{% endcodeblock %}
#### [](#생성자 "생성자")생성자

{% codeblock %}
class <클래스명>:  
 def __init__(self,<추가 매개변수>):  
 ...  
{% endcodeblock %}

클래스명과 이름이 같은 함수를 생성자라고 하며  
내용은 __init__ 이라는 함수에 정의됩니다.

클래스 내부의 존재하는 함수는 반드시 첫번쨰 매개변수로 self를 받습니다.  
self는 자시자신을 나타내는 딕셔너리라고 합니다.

#### [](#소멸자 "소멸자")소멸자

생성자가 있다면 소멸자도 있습니다.  

{% codeblock %}
class <클래스명>:  
 def __del__(self):  
 ...  
{% endcodeblock %}

__del__의 형태로 구현합니다.

생성자는 클래스가 생성될때,  
소멸자는 클래스가 소멸될때 호출됩니다.

##### [](#Example "Example")Example

{% codeblock lang:python %}
class Test:  
 def __init__(self, name):  
 self.name = name  
 print("{} - 생성되었습니다".format(self.name))  
 def __del__(self):  
 print("{} - 파괴되었습니다".format(self.name))  

test = Test("A")  
del test  
{% endcodeblock %}



##### [](#Result "Result")Result

{% codeblock %}
A - 생성되었습니다  
A - 파괴되었습니다  
{% endcodeblock %}

#### [](#메서드 "메서드")메서드

클래스 내에 존재하는 함수를 메서드라고 합니다.  
선언 방식은 생성자,소멸자와 동일합니다.

##### [](#Example-1 "Example")Example

{% codeblock lang:python %}
class Test:  
 def __init__(self, name):  
 self.name = name  
 print("{} - 생성되었습니다".format(self.name))  
 def __del__(self):  
 print("{} - 파괴되었습니다".format(self.name))  
 def sum(self,a,b)  
 print(a+b)  

test = Test("A")  
test.sum(3,5)  
{% endcodeblock %}



##### [](#Result-1 "Result")Result

{% codeblock %}
A - 생성되었습니다  
8  
{% endcodeblock %}
