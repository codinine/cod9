#### [](#파이썬-용어-정리 "파이썬 용어 정리")파이썬 용어 정리

*   `표현식(문장)` - 값을 만들어내는 간단한 코드
*   `키워드` - 파이썬에서 특별한 의미를 부여한 단어

다음과 같은 명령어로 키워드 리스트를 확인할 수 있습니다.  

{% codeblock lang:python %}
import keyword  
print(keyword.kwlist)  
{% endcodeblock %}

*   `식별자` - 이름을 붙일 때 사용하는 단어, 주로 변수 또는 함수에 사용

    ##### [](#식별자-명명-규칙 "식별자 명명 규칙")식별자 명명 규칙

    *   키워드 사용 금지
    *   특수 문자는 _ (언더바)만 허용
    *   숫자로 시작 금지
    *   공백 사용 금지

    ##### [](#스네이크-케이스와-캐멀-케이스 "스네이크 케이스와 캐멀 케이스")스네이크 케이스와 캐멀 케이스

    *   `스네이크 케이스` - 단어 사이에 _ (언더바)를 붙여 식별자를 만든다.  
        ex) item_list, login_status
    *   `캐멀 케이스` - 단어들의 첫 글자를 대문자로 하여 식별자를 만든다.  
        ex) ItemList, LoginStatus
*   `주석` - 프로그램에 영향을 주지 않는 코드. 앞에 #를 붙여 사용.

* * *

#### [](#파이썬-기본-출력 "파이썬 기본 출력")파이썬 기본 출력

![](/images/python-basic-02_1.png)  
그림과 같이 IDLE를 실행하고 `File -> New File`로 파일을 생성하면  
메모장과 같이 Text 편집기가 나옵니다. 여기서 작업을 하시면  
여러 라인을 입력할 수 있고 `F5`로 컴파일 할 수 있습니다.

{% codeblock lang:python %}
# 이것은 주석입니다.  
print("문자를 출력합니다.")  
print('큰 따옴표, 작은 따옴표 모두 사용 가능합니다')  
# 숫자도 출력할 수 있습니다.  
print(10, 20, 30, 40, 50)  
# 아무것도 입력하지 않으면 줄 바꿈이 됩니다.  
print()  
print("print()는 화면에 출력을 해주는 함수입니다")  
#만약 따옴표 안에 또 따옴표를 넣고 싶다면 어떻게 해야될까요?  
print("밖에 큰 따옴표를 썼다면 안에는 '작은 따옴표'를 사용할 수 있습니다")  
print("큰 따옴표 안에 큰 따옴표를 쓰려면 \\"이렇게 하시면 됩니다\\"")  
#역슬래시를 사용하여 표현하는 특수문자를 이스케이프 문자라고 합니다.  
{% endcodeblock %}

#### [](#이스케이프-문자 "이스케이프 문자")이스케이프 문자

일부 특수문자들은 프로그램 언어에서 지정해놓은 기능과 중첩되어  
바로 표현할 수 없는 것들이 있습니다.  
이런 문자들은 역슬래시()를 이용하여 표현할 수 있는데  
이러한 문자들을 이스케이프 문자라고 합니다.  

{% codeblock lang:python %}
print("큰 따옴표 안에 큰 따옴표를 쓰려면 \"이렇게 하시면 됩니다\"")  
print("한줄의 문장을 이스케이프 문자를 이용하여\\n줄바꿈을 할 수도 있습니다")  
print("\t 이 문자는 탭을 의미합니다.")  
print("역슬래시 기호를 사용하고 싶다면 \\ 역슬래시를 하나 더 추가하세요")  
print("""프린트 함수 하나로  
여러 줄의 문자열을 입력하고 싶다면  
이렇게 하면 됩니다.  
큰 따옴표 3개씩 붙이면 되요.  
줄바꿈도 자동으로 됩니다.""")  
{% endcodeblock %}
