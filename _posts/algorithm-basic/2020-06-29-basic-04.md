### [](#분할정복 "분할정복")분할정복

퀵 정렬을 이해하기 위해서는 지난 번에 공부했던 재귀뿐만 아니라  
분할 정복에 대한 이해도 필요합니다.  
분할 정복은 자료구조가 아니라 문제를 해결하는 전략(방법)이라고 할 수 있습니다.

![](/images/algorithm-basic-04_1.jpg)  
가로 1680m , 세로 640m의 토지를 가지고 있다고 가정하겠습니다.  
이 토지를 똑같은 정사각형의 크기로 가능한한 크게 나누어야 합니다.

![](/images/algorithm-basic-04_2.jpg)  
그럼 일단 이 토지로 만들 수 있는 가장 큰 정사각형을 찾아보겠습니다.  
가로 640m, 세로 640m 이면 될 것 같네요.

그리고 남은 영역을 또 최대 크기의 정사각형으로 나눕니다.  
이 일을 반복하여 더 이상 나눌 수 없을때까지 나누었을 때 나오는  
정사각형의 크기, 그것이 이 땅을 똑같은 정사각형으로 나눌 수 있는  
가장 큰 정사각형이 됩니다.

정답은 80m 입니다.  
수학을 기억하신다면 더 쉬운 방법을 알고 있으실 겁니다. 바로 최대 공약수를 구하는 것이지요.
하지만 이런 방법을 사용하지 않는 것은 앞으로 배울 `퀵 정렬`을 이해하기 위해서 입니다.  
위와 같은 방법 역시 재귀의 방법을 사용했다는 것을 알 수 있습니다.

### [](#퀵-정렬 "퀵 정렬")퀵 정렬

원소의 숫자가 무작위로 나열된 배열이 있다고 가정하겠습니다.  
우리는 이 원소르들 중하나를 `기준값`으로 잡습니다.  
그리고 그 기준값과 나머지 값들을 하나하나 비교하며 큰지, 작은지 파악합니다.  
그러면 이렇게 정리가 될겁니다.  
작은 수 , `기준값` , 큰 수

그럼 작은 수의 그룹에서 다시 하나의 기준값을 설정하고 위와같은 행위를 반복합니다.  
큰 수의 그룹에서도 마찬가지로요. 그럼 최종적으로 순서대로 정렬이 완료될 것입니다.

이제 예를 들어보겠습니다.  
list = [30,15,22,8,51]  
여기서 15를 기준값으로 잡아보겠습니다.  
작은 수 그룹(8) , 기준 값(5), 큰 수 그룹(30,22,51)  
작은 수 그룹은 더 이상 정렬할 값이 없으므로 큰 수 그룹에서 하나를 선택하겠습니다.  
51을 선택했습니다.  
작은 수 그룹(30,22) , 기준값(51), 없음  
마지막으로 남은 30과 20을 비교합니다.  
그럼 결과적으로 list = [8,15,22,30,51] 의 결과를 얻게 됩니다.  

이제 코드로 구현해보도록 하겠습니다.

#### [](#구현 "구현")구현

##### [](#Example "Example")Example

```python
def quicksort(array):  
 # 길이가 2보다 작다면 그대로를 리턴합니다.  
 if len(array) < 2:  
 return array  
 else:  
 pivot = array[0]  
 # 배열의 첫번째 값을 기준으로 하여 작은 값은 less에  
 # 큰값은 greater에 넣습니다.  
 less = [i for i in array[1:] if i <= pivot]  
 greater = [i for i in array[1:] if i > pivot]  
 #less와 greater는 재귀로 다시 호출합니다.  
 return quicksort(less) + [pivot] + quicksort(greater)  

print(quicksort([10, 5, 2, 3]))  
```

##### [](#Result "Result")Result

```
[2, 3, 5, 10]  
```
